GOT QUESTIONS?

Who built this?

SANDWICH was built at Etalas, an Indonesian software house working with enterprise clients. These skills were forged from actual agency battles — messy inputs, unpredictable AI, and the need for reliable delivery.

Why does this exist?

Client briefs are chaotic by nature. SANDWICH gives you a repeatable pipeline — from raw client input to validated, machine-checkable specs — so your AI agent doesn't guess, it executes.

Is it free?

Yes. Fully open source on GitHub. Install it with one command and use it in any project, no account required.

Which AI agents are supported?

Pi, Claude Code, and Codex. Each has a one-liner install command — pick your agent from the hero and copy it. More agents coming as the ecosystem grows.

Do I need to configure anything?

Just install via your agent's skill manager. The Order → Prep → Recipe pipeline works out of the box. Each ingredient is a standalone skill — use all three or drop in just what you need.
